  1. About The Application

This application is to be used by buy-here-pay-here car dealerships. It is used to keep track of customers who have bought insurance, or who need to provide proof of insurance or buy insurance. It also keeps a list of lenders and their requirements for insurance. It also allows for information on the employees using it to be stored, and it allows for customers to provide their insurance, or be redirected to buy it from WeCare.

Now this application can only be run locally after you get the code from this here and view the effect at http://localhost:8080/lender/home.html. But the application will be able to visit from public internet after we finish development and deploy it on oracle cloud.
